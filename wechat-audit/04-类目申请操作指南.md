# 类目申请操作指南

> 目的：在微信小程序后台添加"深度合成-AI问答"服务类目
> 前置条件：算法备案截图 ✅ + 腾讯云订单合同 ✅
> 预计耗时：提交10分钟，审核1-3个工作日

---

## 前置条件确认

在开始申请前，请确认已获取以下材料：

- [ ] 算法备案截图（参考 [02-算法备案截图获取指引](./02-算法备案截图获取指引.md)）
- [ ] 腾讯云混元订单合同（参考 [03-腾讯云混元开通指引](./03-腾讯云混元开通指引.md)）
- [ ] 主体一致性已确认（参考 [05-主体一致性核对清单](./05-主体一致性核对清单.md)）

---

## 操作步骤

### Step 1：登录微信公众平台

```
https://mp.weixin.qq.com
```

使用小程序管理员微信扫码登录。

> ⚠️ 确保登录的是**小程序**账号，不是公众号。小程序AppID：wx97425a7556eb8572

### Step 2：进入服务类目设置

路径：
```
设置 → 基本设置 → 服务类目
```

操作步骤：
1. 点击左侧菜单 **"设置"**
2. 点击 **"基本设置"**
3. 找到 **"服务类目"** 区域
4. 点击 **"添加类目"** 或 **"修改"**

### Step 3：选择目标类目

在服务类目选择页面中：

1. 一级类目选择：**"深度合成"**
2. 二级类目选择：**"AI问答"**（或"智能问答"）

> 💡 如果找不到"深度合成"类目，可能需要先完成主体认证或更新小程序版本。

### Step 4：上传资质材料

系统会要求上传以下材料：

| 材料 | 上传内容 | 格式要求 |
|------|----------|----------|
| 算法备案证明 | 算法备案截图 | PNG/JPG，清晰可读 |
| 合作协议 | 腾讯云订单合同 | PDF，完整合同 |

#### 上传注意事项

- 截图要清晰，关键信息（备案编号、算法名称、类型）必须可辨认
- 合同文件须为完整版，不要裁剪
- 文件大小不超过平台限制（通常5MB以内）
- 建议提前将文件命名清晰，如"算法备案截图.png"、"腾讯云混元合同.pdf"

### Step 5：填写补充信息

可能需要填写：

| 字段 | 建议填写 |
|------|----------|
| 服务说明 | 基于AI技术的餐饮经营管理智能问答服务 |
| 应用场景 | 餐饮店经营数据查询、成本核算、营销文案生成、评价分析等 |
| 使用的大模型 | 腾讯混元大模型 |
| 算法备案编号 | 网信算备440305295988701230071号 |

### Step 6：提交审核

1. 确认所有信息填写完整
2. 确认材料上传成功
3. 点击 **"提交"**
4. 记录提交时间

### Step 7：等待审核

| 阶段 | 预计时间 |
|------|----------|
| 提交成功 | 即时 |
| 初审 | 1个工作日 |
| 终审 | 1-3个工作日 |
| 总计 | 1-3个工作日 |

审核结果会通过微信消息通知管理员。

---

## 审核可能的结果

### ✅ 通过
- 类目成功添加
- 可以开始接入AI能力

### ❌ 被拒（常见原因及应对）

| 被拒原因 | 应对方案 |
|----------|----------|
| 截图不清晰 | 重新截图，确保关键信息可辨认 |
| 合同信息不匹配 | 确认合同主体与小程序主体一致 |
| 缺少算法备案编号 | 在补充说明中标注编号 |
| 类目选择错误 | 确认选择的是"深度合成-AI问答"而非其他 |
| 主体认证未完成 | 先完成小程序主体认证 |

被拒后可修改材料重新提交，无次数限制。

---

2. 配置 SKILL.md 和 mcp.json
3. 开发原子接口
4. 测试联调
