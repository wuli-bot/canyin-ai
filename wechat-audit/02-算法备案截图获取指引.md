# 算法备案截图获取指引

> 目的：从网信办算法备案系统获取腾讯混元大模型的算法备案截图
> 预计耗时：10分钟

---

## 操作步骤

### Step 1：访问算法备案系统

打开浏览器，访问：
```
https://beian.cac.gov.cn
```

> 这是国家互联网信息办公室的"互联网信息服务算法备案系统"

### Step 2：查询算法备案

在备案系统的查询页面中：

1. 选择查询方式：**按备案编号查询**
2. 输入备案编号：
   ```
   网信算备440305295988701230071号
   ```
3. 点击"查询"

> 💡 这个编号是腾讯混元大模型的"生成合成类（深度合成）"算法备案编号，已有开发者验证过审成功。

### Step 3：确认备案信息

查询结果应显示以下关键信息（核对确认）：

| 字段 | 期望值 |
|------|--------|
| 算法名称 | 腾讯混元大模型（或类似名称） |
| 算法类型 | 生成合成类（深度合成） |
| 备案主体 | 深圳市腾讯计算机系统有限公司（或腾讯云相关主体） |
| 备案编号 | 网信算备440305295988701230071号 |
| 状态 | 有效/已备案 |

### Step 4：截图保存

对查询结果页面进行**完整截图**，确保以下信息清晰可见：

- [ ] 备案编号（完整显示）
- [ ] 算法名称
- [ ] 算法类型（须含"生成合成类"或"深度合成"）
- [ ] 备案主体
- [ ] 备案状态
- [ ] 页面URL（地址栏显示 beian.cac.gov.cn）

**截图要求**：
- 全屏截图，不要裁剪
- 分辨率清晰（建议1920×1080以上）
- 包含浏览器地址栏（证明来源）
- 保存为 PNG 或 JPG 格式

### Step 5：文件命名

将截图文件命名为：
```
算法备案截图_腾讯混元.png
```

---

## 备用方案

如果 beian.cac.gov.cn 无法访问或查询不到结果：

1. **方式B**：在搜索引擎搜索"网信算备440305295988701230071号"，找到第三方转载的备案信息页面并截图
2. **方式C**：访问腾讯云官方文档中关于混元大模型算法备案的说明页面截图
3. **方式D**：联系腾讯云客服，请求提供算法备案证明
