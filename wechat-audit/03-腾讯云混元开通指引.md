# 腾讯云混元开通指引

> 目的：开通腾讯混元大模型服务 + 获取订单合同（合作协议）
> 预计耗时：开通5分钟，合同申请1-2个工作日

---

## 前置条件

- [ ] 已有腾讯云账号（没有的话先去 https://cloud.tencent.com 注册）
- [ ] 腾讯云账号已完成实名认证（个体工商户/企业认证）

> ⚠️ 腾讯云实名认证主体须与小程序主体一致（长沙市望城区周兰英餐饮店）

---

## 第一部分：开通腾讯混元大模型服务

### Step 1：登录腾讯云控制台

```
https://console.cloud.tencent.com
```

### Step 2：进入混元大模型产品页

方式一：控制台搜索"混元"或"Hunyuan"
方式二：直接访问
```
https://console.cloud.tencent.com/hunyuan
```

### Step 3：开通服务

1. 首次进入会提示"开通服务"或"立即开通"
2. 点击开通
3. 阅读并同意服务协议
4. 确认开通

> 💡 开通本身免费，按实际调用量计费。初期可用免费额度测试。

### Step 4：获取 API Key

1. 进入混元控制台
2. 找到"密钥管理"或"API密钥"
3. 创建一个新的 API Key
4. **立即保存**（部分页面只显示一次）

```
# API Key 示例格式（实际以控制台为准）
SK-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

> ⚠️ API Key 请妥善保管，不要泄露。后续开发会用到。

### Step 5：产生订单（关键步骤）

要申请合同，需要有一笔实际订单（哪怕是1元钱的测试调用）。

1. 在混元控制台发起一次测试调用
2. 使用 API 调用任意模型（如 hunyuan-lite）
3. 产生一笔费用（哪怕只有几分钱）

> 💡 调用示例见本文末尾"附录：混元API调用示例"

---

## 第二部分：申请订单合同

### Step 6：进入费用中心

```
https://console.cloud.tencent.com/expense/overview
```

路径：控制台 → 费用中心

### Step 7：申请合同

1. 点击左侧菜单 **"订单合同"** 或 **"合同管理"**
2. 点击 **"申请合同"** 或 **"新建合同"**
3. 选择合同类型：**订单合同**
4. 勾选产品：找到并勾选 **"腾讯混元大模型"** 或 **"AI与机器学习"** 相关产品
5. 填写合同信息：
   - 合同主体：长沙市望城区周兰英餐饮店
   - 联系方式：填写你的信息
6. 提交申请

### Step 8：等待合同生成

- 电子合同通常 **1-2个工作日** 生成
- 生成后在"合同管理"页面可下载
- 合同类型为：**电子合同（PDF）**

### Step 9：下载并确认合同内容

下载合同后，确认以下关键信息：

- [ ] 合同包含"腾讯混元大模型"或相关AI服务名称
- [ ] 合同主体为：长沙市望城区周兰英餐饮店
- [ ] 合同编号清晰可见
- [ ] 合同有效期在有效期内

## 附录：混元API调用示例

### cURL 调用

```bash
curl -X POST "https://hunyuan.tencentcloudapi.com" \
  -H "Content-Type: application/json" \
  -H "X-TC-Action: ChatCompletions" \
  -H "X-TC-Version: 2023-09-01" \
  -H "Authorization: TC3-HMAC-SHA256 Credential=AKIDxxxxxxxx/..." \
  -d '{
    "Model": "hunyuan-lite",
    "Messages": [
      {"Role": "user", "Content": "你好，请介绍一下自己"}
    ]
  }'
```

### Python 调用

```python
import json
import requests

# 建议使用腾讯云官方SDK
# pip install tencentcloud-sdk-python

from tencentcloud.common import credential
from tencentcloud.hunyuan.v20230901 import hunyuan_client, models

cred = credential.Credential("SecretId", "SecretKey")
client = hunyuan_client.HunyuanClient(cred, "")

req = models.ChatCompletionsRequest()
req.Model = "hunyuan-lite"
req.Messages = [
    {"Role": "user", "Content": "你好"}
]

resp = client.ChatCompletions(req)
print(json.loads(resp.to_json_string()))
```
