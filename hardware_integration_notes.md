## 一、小爱同学（语音入口）

### 接口信息
- **平台**：小米开放平台 (https://dev.mi.com)
- **能力**：自定义语音技能（Custom Skill）
- **协议**：小米IoT技能协议 / HTTP Webhook
- **对接思路**：
  1. 在小米开放平台注册自定义技能
  2. 配置Webhook URL指向MCP Server（需公网地址）
  3. 用户对小爱说"问一下店长今天营业额" → 小爱转发到Webhook → MCP查询Supabase → 返回结果 → 小爱语音播报
- **依赖**：MCP Server需公网部署（腾讯云轻量服务器）
- **备选方案**：使用Coze语音总控Bot（7652253691269103651），通过手机端语音输入

### 待确认
- [ ] 小爱同学设备型号（是否支持自定义技能）
- [ ] 小米开放平台账号是否已注册
- [ ] 门店WiFi网络稳定性

## 二、树莓派（边缘计算节点）

### 接口信息
- **系统**：Raspberry Pi OS (Linux)
- **环境**：Python 3.x + 网络连接
- **用途**：
  1. 本地数据采集（传感器/摄像头）
  2. 轻量级API网关（局域网内设备统一出口）
  3. MCP Server本地代理（减少云依赖）
- **对接思路**：
  1. 树莓派安装Python + FastAPI
  2. 运行轻量级数据采集服务（温度/湿度/人流量）
  3. 通过HTTP/WebSocket推送数据到Supabase
  4. 可选：本地运行MCP Server副本，降低延迟
- **依赖**：树莓派硬件、传感器模块（如有）
- **网络**：需连接门店WiFi，能访问互联网

### 待确认
- [ ] 树莓派型号（3B/4B/Zero）
- [ ] 是否已安装Python环境
- [ ] 传感器配置（如有）

## 三、摄像头（图像识别）

### 接口信息
- **用途**：菜品识别 / 客流统计 / 后厨监控
- **协议**：RTSP流（网络摄像头）或 USB（本地摄像头）
- **对接思路**：
  1. 摄像头接入树莓派或门店电脑
  2. 使用OpenCV/YOLO做图像识别
  3. 识别结果（出餐计数/客流量）写入Supabase
  4. AI店长可查询实时数据
- **备选方案**：使用手机拍照 + AI识别（轻量级方案）
